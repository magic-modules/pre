@magic-modules/pre

this is the @magic-modules Pre component. It provides syntax highlighting for javascript

installation:

installation is done using npm. for now, all magic modules are living on github and not on npm.

note the missing @ before magic-modules.

this is how we install npm modules from github.

npm install magic-modules/pre

require:

first add the component to the assets

// assets/index.jsmodule.exports = {  //...other exports  Pre: require('@magic-modules/pre'),}

usage:

in a page/component, just pass some string that looks like js

module.exports = {  View: () => Pre.View('const js = true'),}

renders

const js = true

fixed color:

you can give a fixed color to a Pre.View by passing a second argument with the color.

as you can see below, this also removes the theme choose buttons

dark

Pre.View('content', 'dark')

light

Pre.View('content', 'light')
